## 5. 接口详情

### `GET /api/example`

用途：

- 描述这个接口在业务流程中的作用

请求参数：

| 参数 | 位置 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
| `id` | `query` | `String` | 否 | 示例 |

成功响应：

- `200 OK`

```json
{
  "id": "example-1"
}
```

错误响应：

| HTTP 状态 | 错误体 | 说明 |
| --- | --- | --- |
| `400` | `ErrorResponse` | 参数不合法 |
| `404` | `ErrorResponse` | 资源不存在 |

兼容性约束：

- 新增字段向后兼容
- 删除字段前先标记弃用
